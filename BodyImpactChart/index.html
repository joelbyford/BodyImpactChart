HTML5 Body Impact Chart Plugin

Simple body impact chart clicker. Provided in both pure JavaScript and as a stateful jQuery widget (using the jQuery Widget Factory model). Both samples should function the same.

Pure JavaScript

  • HTML Example - PureJavascript/javascriptVersion.html
  • Body Background Image - body.png

jQuery Widget (Preferred)

  • HTML Example - JQueryWidget/jQueryVersion.html
  • JavaScript Include - JQueryWidget/BodyImpactWidget.js
  • Body Background Image - body.png
  • jQuery Include - Any Compatible jQuery 1.8+ Library downloaded from jQuery.com
  • jQuery-UI Include - Any Compatible jQuery-UI 1.8+ Library downloaded from jQueryUI.com
